# Snake and Ladder Game

## 📝 About the Project
The Snake and Ladder Game is a beginner-friendly C application that simulates the classic board game in a console environment. The application allows two players to roll a dice, move across the board, climb ladders, slide down snakes, and compete to reach the final position.

## ✨ Features
- Two-player gameplay
- Random dice roll simulation
- Snake and ladder movement
- Player position tracking
- Win detection
- Simple console-based interface

## 🛠️ Technologies Used
- C Programming
- Arrays
- Functions
- Loops
- Conditional Statements
- Random Number Generation

## 🎯 Learning Outcomes
This project helped me learn:
- Working with random number generation
- Implementing game logic
- Using functions for modular programming
- Managing player positions
- Developing interactive console applications
